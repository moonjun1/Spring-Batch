🚀 Spring Batch 애플리케이션

CSV 파일을 읽어서 데이터베이스에 저장하는 배치 처리 시스템

📋 사용 방법

  1. 배치 실행: CSV 파일의 사용자 데이터를 데이터베이스에 저장합니다.
  2. 데이터 확인: 저장된 데이터를 JSON 형태로 확인할 수 있습니다.
  3. H2 콘솔: SQL을 직접 실행하여 데이터를 조회할 수 있습니다.

🎯 메뉴

📊 저장된 사용자 데이터 🌦️ 날씨 대시보드 🗄️ H2 데이터베이스 콘솔

🔧 H2 콘솔 접속 정보

  • JDBC URL: jdbc:h2:mem:testdb
  • 사용자명: sa
  • 비밀번호: (비워두세요)
